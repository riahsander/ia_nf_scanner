Riah.AI - Sistema de Gestão de Notas Fiscais com IA

Frontend: Laravel Blade, Tailwind CSS e JavaScript (Fetch API para modais).

Backend: PHP 8.3 + Laravel 13, utilizando a biblioteca Carbon para tratamento rigoroso de datas.

Inteligência Artificial: Microserviço em Python rodando Tesseract OCR para leitura de imagem e Groq (LLM) para estruturação de dados em JSON.

Infraestrutura: Arquitetura baseada em Docker, isolando o serviço de IA do servidor web para maior performance e escalabilidade.

Diferencial: Blindagem de dados contra corrupção de JSON e tratamento automático de diferentes formatos de notas fiscais brasileiras.
